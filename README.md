# UCALL India - Premium Leather E-Commerce Website

## 🛍️ Overview

UCALL India is a modern, responsive e-commerce website built with HTML, CSS, and JavaScript. It features a premium leather products collection with a complete shopping experience including product browsing, cart management, user authentication, and checkout functionality.

## ✨ Features

### 🏪 Core Features
- **Product Catalog**: Browse a curated collection of 12+ premium leather products
- **Product Categories**: Filter products by bags, wallets, belts, and accessories
- **Product Details**: View ratings, reviews, prices, and product information
- **Shopping Cart**: Add/remove items, update quantities, view cart total
- **Checkout Process**: Complete order with shipping and payment options
- **Order Tracking**: Get order confirmation and ID

### 👤 User Management
- **Sign Up**: Create a new account with validation
- **Login**: Secure user login functionality
- **Local Storage**: Persistent cart and user data
- **User Profile**: Store customer information

### 💳 Payment & Checkout
- **Multiple Payment Methods**:
  - Credit/Debit Card
  - UPI Payment
  - Net Banking
  - Cash on Delivery
- **Shipping Calculation**: Free shipping on orders over ₹2000
- **Tax Calculation**: Automatic 18% GST calculation
- **Order Summary**: Detailed breakdown of costs

### 🎨 Design Features
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Premium leather-themed color scheme
- **Smooth Animations**: Microinteractions and transitions
- **Dark Navigation**: Professional gradient navbar
- **Product Badges**: Sale, Hot, New product indicators
- **Hover Effects**: Interactive product cards and buttons

### 📱 Additional Features
- **Search Functionality**: Search bar for product discovery
- **Newsletter Signup**: Email subscription for updates
- **Customer Testimonials**: 5-star reviews from verified customers
- **Feature Highlights**: Fast shipping, secure payment, easy returns, 24/7 support
- **Social Links**: Connect via Facebook, Instagram, Twitter, LinkedIn
- **FAQ & Support**: Customer service information

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/zuhair-neyer/ucall-india.git
```

2. Navigate to the project directory:
```bash
cd ucall-india
```

3. Open `index.html` in your web browser:
```bash
open index.html
# or
start index.html
```

That's it! The website will run locally without any installation.

## 🎮 Usage

### Browsing Products
1. Navigate to the "Products" section
2. Use filter buttons to browse by category
3. View product details, prices, and reviews

### Shopping
1. Click "Add to Cart" on any product
2. View your cart by clicking the shopping bag icon
3. Update quantities or remove items
4. Click "Proceed to Checkout"

### Creating Account
1. Click "Sign Up" button
2. Fill in your details (Name, Email, Phone, Password)
3. Accept terms and create account
4. Account data is saved locally

### Checkout
1. Review your cart items
2. Enter shipping address
3. Choose payment method
4. Complete payment (demo)
5. Receive order confirmation

## 💾 Local Storage

The application uses browser localStorage to persist:
- **Cart Items**: Saved with quantities and totals
- **User Data**: Account information (email, name, phone)
- **Last Order**: Order history and order ID

Clear browser cache to reset all data.

## 🔐 Security Notes

This is a **demo/frontend application**:
- Payments are simulated (no real transactions)
- User data is stored locally in browser
- No backend server or database
- For production, integrate with:
  - Real payment gateway (Razorpay, Stripe, PayPal)
  - Backend server for data persistence
  - Database for user and order management
  - SSL/TLS encryption
